# Elite Championship

This repository now contains a React + Vite + Tailwind implementation of the Elite Championship site.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Project structure

- `index.html` — Vite entry point
- `src/main.jsx` — React application bootstrap
- `src/App.jsx` — app shell, router, and footer
- `src/data.js` — site data and leaderboard records
- `src/pages/` — page components for Home, Leaderboard, Compare, Auctions
- `src/components/` — shared components, including Navbar, StarField, Countdown, Marquee
- `src/index.css` — Tailwind base and custom theme styling
